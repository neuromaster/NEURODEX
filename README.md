# NEURODEX $NeuroDEX
NEURODEX #NEUROMASTER

Custom DEX, NFT, CRYPTO CAPACITOR, #NeuroDEX 

NEUROMASTER is a decentralised financial payment network that rebuilds the traditional payment stack on the blockchain. It utilises a basket of fiat-pegged stable-coins SOLANA, BSC, WAVES and  CRYPTONOTE, algorithmically stabilised by its reserve currency $NXR + $NEUROMASTER, to facilitate programmable payments and open financial infrastructure development. December 2022, the network has transacted an estimated $6 Trillion for over 5 million users.
